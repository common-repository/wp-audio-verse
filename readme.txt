== Description ==

The WP Audio Verse is a WordPress Plugin which helps to publish online book viewer with audio. Publish Holy Quran with Audio and translation facilities. Also, its help to publish holy bible, verse, sermons, lyrics, poem, spoken english publish, online english etc.

<h2>Features</h2>
<ul>
<li>Display Online different kind of book with translation facilities </li>
<li>Publish Online eBook</li>
<li>Publish Holy Quran Surah with audio and translation facilities</li>
<li>Publish Holy Muslim religion verse and verses with mp3</li>
<li>Publish Christian religion verse  with mp3</li>
<li>Publish Hindu religion verse</li>
<li>Publish buddhist religion ebook</li>
<li>Publish Lyrics with translation facilies with mp3</li>
<li>Publish Poem with translation facilities</li>
</ul>

== Installation ==
1. Upload `wp-audio-verse.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Add new verse and see the settings option below of the container.
